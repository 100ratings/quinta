Quick rules & reminders

Core of the method

• Force 1 item among 5; the force item is always the 2nd from the left (fixed).
• Type: if N is even → Basic; if N is odd → Bounce.
• Side (Basic): start on the left if N ≡ 0 or 2 (mod 8), otherwise on the right.
• Side (Bounce): use N+1 with the same rule above (and count moves).
• “Basic” counts points 1,2,3,4,5,4,3,2,1…; “Bounce” counts moves.

Ensuring even (optional)

• IBO/DBO (increase/decrease by one).
• Others: “tag team”, pairing, selection field with even totals, etc.

Tips

• Shortcuts: N, Enter, H.
• Think only about N mod 8 + parity.
• “Bounce” with a moving piece (e.g., a pawn) justifies the moves.

Mental example

N=22 → Basic; 22 ≡ 6 (mod 8) ⇒ start on the Right.
N=33 → Bounce; use 34 ⇒ 34 ≡ 2 (mod 8) ⇒ start on the Left.
